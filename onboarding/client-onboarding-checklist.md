# Client Onboarding Checklist — Growth-onomics
*Works for any client, any vertical. Fill in context, assign to Atlas to track.*

---

## PHASE 1 — Intake (Day 1)

### Business Context
- [ ] Company name, website, industry/vertical
- [ ] Target audience (who are we marketing to?)
- [ ] Geographic focus (local, national, global?)
- [ ] Main products/services being promoted
- [ ] Key differentiators / USPs
- [ ] Top 3 competitors (for benchmarking)
- [ ] Current monthly traffic / leads / revenue (baseline)

### Goals & KPIs
- [ ] Primary goal (leads? sales? brand awareness? rankings?)
- [ ] Target KPIs for 90 days
- [ ] What does success look like at 6 months?
- [ ] Any hard constraints (budget caps, brand guidelines, topics to avoid)

### Services Scoped
- [ ] SEO — confirm target keyword clusters
- [ ] Paid Ads — platforms (Google/Meta/LinkedIn), budget, target CPA
- [ ] AEO — target queries for AI answer engines
- [ ] Content — formats, cadence, approval process

---

## PHASE 2 — Access & Setup (Days 1-3)

### Access Required
- [ ] Google Analytics 4 access (Editor level)
- [ ] Google Search Console access (Owner or Full User)
- [ ] Google Ads access (if Paid Ads in scope)
- [ ] Meta Ads Manager (if Meta in scope)
- [ ] Google Business Profile (if local SEO in scope)
- [ ] CMS access (for on-page SEO implementation)
- [ ] Looker Studio — create client dashboard, share view link

### Technical Baseline
- [ ] Crawl site (Screaming Frog / Ahrefs) — document technical issues
- [ ] Baseline keyword rankings (export to sheet)
- [ ] Baseline traffic (GA4 — last 3 months)
- [ ] Baseline ad performance (if taking over existing campaigns)

---

## PHASE 3 — Strategy Brief (Days 3-7)

### Deliverables
- [ ] Context brief created (1-pager: client background, audience, goals, tone of voice)
- [ ] Keyword strategy document — target clusters, priority keywords, gap analysis
- [ ] Content plan — first 30 days (topics, formats, owner)
- [ ] Paid Ads plan (if applicable) — campaign structure, targeting, budget allocation
- [ ] AEO opportunity map (if applicable) — top queries to target

### Internal Handoff
- [ ] Context brief saved in Mission Control (tag with client name)
- [ ] Assigned to relevant agents (Hermes for content, Atlas for reporting, Athena for strategy)
- [ ] First reporting period start date confirmed
- [ ] Reporting template set up for this client

---

## PHASE 4 — Client Kickoff

### Communication
- [ ] Kickoff call/email completed
- [ ] Reporting cadence confirmed (bi-weekly default)
- [ ] Client contact confirmed (who receives reports?)
- [ ] Response time expectations set (client side)
- [ ] Approval process agreed (do they need to sign off on content before publish?)

### Client Deliverables
- [ ] Looker Studio dashboard link shared
- [ ] First report date communicated
- [ ] Emergency contact established (who to ping if something goes wrong)
